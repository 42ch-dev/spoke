# SPOKE core protocol bootstrap ‚Äî Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `mstar-sdd` (recommended) or inline execution. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Stand up SPOKE JSON Schema SSOT for **data + ops** layers, generate `@42ch/spoke-schema` (TS) and `spoke-schema` (Rust), leave **empty adapter directories**, and **skip fixtures**.

**Architecture:** `schemas/` is the only hand-authored wire truth. Codegen emits TypeScript (`@42ch/spoke-schema`) and Rust (`spoke-schema`) packages under mirrored `generated/` trees. Protocol objects carry a required `extensions` map for product-specific payloads so future adapters can round-trip without loss. Behavioral ops are separate schema families from data envelopes. **`Rule` is deferred** in v0.1 (no `rule.schema.json`); `assemble` is wire-only (`AssemblePacket` shape, no compute fields).

**Tech Stack:** JSON Schema (draft-07 compatible), `json-schema-to-typescript`, `typify`, pnpm workspace (`packages/*`, `tooling/*`), Cargo workspace (`crates/spoke-schema`).

**Architecture SSOT:** [`specs/spoke-protocol.md`](../../specs/spoke-protocol.md) (codegen layout), [`specs/spoke-data-model.md`](../../specs/spoke-data-model.md) (5 data objects + `Rule` deferral), [`specs/spoke-ops.md`](../../specs/spoke-ops.md) (5 ops + assemble boundary), [`schemas/README.md`](../../schemas/README.md) (17-file tree).

**Execution:** mstar-sdd

## Global Constraints

- Two layers only for v0.1 normative surface: **data** + **ops**
- Package name is **`spoke-schema`** (not contracts)
- `extensions.<namespace>` is the sole product-specific bag; core objects `additionalProperties: false`
- Adapters: `adapters/nexus/` + `adapters/creader/` empty (`.gitkeep`) ‚Äî no conversion code
- **No fixtures** ‚Äî no `fixtures/`, golden files, or conformance harness
- Prefer third-party codegen; do not invent a custom codegen engine
- Do not default-publish to npm/crates.io

### Architecture decisions (architect, 2026-07-23)

| Decision | Choice | Schema impact |
|----------|--------|---------------|
| `Rule` object | **Deferred** v0.1 | No `rule.schema.json`; `check` may use opaque `rule_refs: string[]` |
| `assemble` | Wire-only | `AssemblePacket` data schema + request/response; no ranking/retrieval fields |
| Open vocabulary | `type: string` + documented core list | No `enum` on `block_type` / status in v0.1 |
| Extensions | Required `extensions: {}` on all data objects | `common.schema.json#/definitions/ExtensionMap` |
| Codegen mirror | `generated/{common,data,ops}` in both packages | `tooling/codegen/` orchestrates jstt + typify |
| URI base | `https://spoke42.invalid` | All `$id` / `$ref` values |
